## 代码结构概要

---

### org.xiao.algs包模块列表

---

- **bag：**封装背包数据结构（数组和链表两种实现）

- **io：**封装基本输入输出库

- **queue：**封装队列数据结构（数组和链表两种实现）

- **search：**封装常用查找算法

- **sort：**封装常用排序算法（插入、选择、希尔、归并、快速、堆排序以及一些排序的改进版本）

- **stack：**封装栈数据结构

- **test：**测试包

- **tree：**封装常用树和图算法结构

- **util：**封装算法使用到的工具类
